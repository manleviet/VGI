Dans ce répertoire, il comprend des éléments suivants :
+ Code-source :
	- VGI.zip : le logiciel VGI
	- ConfDatabase.zip : pour créer le fichier de configuration de la base de données
	- VGIinput.zip : pour créer le fichier de configuration vgi version 1.0
	- VisualWordViewer.zip : un logiciel qui peut dessiner les mots visuels d'une image
	- DrawVisualWord.zip : traverser sur un répertoire des images pour dessiner les mots visuels
+ Database.zip :
	- le répertoire BW : il y a deux répertoires Wang-MotVisuel (les images avec les mots visuels) et Wang-ppm (les images pas les mots visuels). Il y a encore un fichier db.conf.
	- database01 : comprend deux répertoires MS et SA. Ce sont les données du logiciel DOAN Tuan Anh
	- cluster# : les données partitionnement du cluster #
+ Documents.zip : contenir les références du TPE
+ Rapport.zip : les rapports et les slides d'exposé. Les fichiers .pages et .keynote sont les fichiers de logiciel iWork qui est un logiciel sur MAC OS X.
+ cluster.ods : une analyse de resultats de la partitionnement de DOAN Tuan Anh
